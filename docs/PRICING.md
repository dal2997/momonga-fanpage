## Free 플랜 (기본)

무료로 제공되며 가입 즉시 사용 가능합니다.

### 포함 기능
- 수집 아이템 최대 **50개** 등록
- 아이템당 이미지 1장 (Supabase Storage 총 50MB)
- 공개 프로필 페이지 (`momonga.app/u/[handle]`)
- 공식/출처불명 태그
- 콜라쥬 생성 및 SNS 공유 (momonga.app 워터마크 포함)
- 커스텀 아바타 업로드
- 초대 코드 발급 및 공유

### 제한
- 아이템 51번째 등록 시 업그레이드 안내 표시
- 이미지 용량 50MB 초과 시 업로드 차단
- CSV/JSON 데이터 내보내기 불가
- 콜라쥬 이미지에 워터마크 자동 삽입 (제거 불가)

## Pro 플랜

개인 컬렉터를 위한 올인원 굿즈 트래커.

### 가격
- **월간 결제**: ₩4,900 / 월
- **연간 결제**: ₩3,900 / 월 (연 ₩46,800 — 약 20% 절약)
- 첫 달 무료 체험 제공 (카드 등록 필요)

### 포함 기능 (Free 플랜 포함 + 아래 추가)
- 수집 아이템 **무제한** 등록
- 이미지 스토리지 **2GB** (아이템당 최대 10MB)
- **CSV / JSON 내보내기** — 백업 및 스프레드시트 연동
- **콜라쥬 워터마크 제거** 옵션
- **고급 태깅**: 시리즈명, 에디션, 컨디션 (민트/사용감/개봉), 취득 경로 등
- **가격 통계 대시보드**: 카테고리별 지출 합계, 월별 추이
- **우선 이메일 지원** (평균 응답 24시간 이내)

## 과금 정책

### 결제
- Stripe를 통한 카드 결제 (국내 카드 포함)
- 자동 갱신 (해지 전까지 매월/매년 청구)

### 업그레이드 / 다운그레이드
- 언제든지 Pro → Free 다운그레이드 가능
- 다운그레이드 시 남은 기간은 유지, 다음 갱신일에 반영
- Free 한도(50개)를 초과한 아이템은 읽기 전용으로 보존 (삭제 안 됨)

### 환불 정책
- 결제 후 **7일 이내** 전액 환불 (미사용 조건)
- 7일 이후 환불 불가 (나머지 기간 사용 가능)

### 학생 할인
- 교육기관 이메일(`.ac.kr`, `.edu`)로 가입 시 Pro 30% 할인 적용
- 인증은 이메일 도메인 자동 검증
